## 🎯 Your Next Steps

### Step 1: Install (5 min)
```bash
npm install
```

### Step 2: Add Images (30 min)
Put your images in:
- `public/images/hero/` (4 slider images)
- `public/` (social sharing image, icons)

See: `public/images/hero/README.md`

### Step 3: Update Content (20 min)
Edit these files:
- `src/data/config.ts` (contact info, links)
- `src/data/testimonials.ts` (student reviews)
- `src/data/courses.ts` (course details)

See: `CONTENT-GUIDE.md`

### Step 4: Test Locally (10 min)
```bash
npm run dev
```
Check everything at http://localhost:3000

### Step 5: Build (5 min)
```bash
npm run build
```
Creates production files in `/out`

### Step 6: Deploy! (30-60 min)
Choose your platform:
- **Netlify** (easiest - recommended)
- **Vercel** (Next.js optimized)
- **GitHub Pages** (free)
- **Your own hosting**

See: `DEPLOYMENT.md` for step-by-step guide
