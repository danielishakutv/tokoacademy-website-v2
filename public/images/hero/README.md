## Image Optimization Tips

For optimal performance in low-bandwidth regions:

1. **Use WebP format** when possible for better compression
2. **Compress images** to reduce file sizes without sacrificing quality
3. **Provide multiple sizes** using Next.js Image component srcset
4. **Use progressive JPEGs** for photos
5. **Optimize hero images** - they're critical for first impression

## Recommended Tools

- TinyPNG (https://tinypng.com) - For PNG compression
- Squoosh (https://squoosh.app) - For WebP conversion and optimization
- ImageOptim (Mac) or FileOptimizer (Windows) - Batch optimization
